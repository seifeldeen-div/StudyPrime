## 🇬🇧 English

Overview

StudyPrime is a web app that helps students build deep focus habits using the Pomodoro technique, combined with a to-do list, study goals tracker, and quick notes — all in one simple, distraction-free interface.


"Master Your Time, Master Your Future."



 ## Features


 Pomodoro Timer — structured focus sessions (default 25:00) with Start/Stop control
 To-Do List — add tasks and track completion with a live progress percentage
 Study Goals — set and track personal study goals
 Quick Notes — jot down notes, auto-saved to local storage
 Phone-style Web App UI — clean, mobile-frame interface for a focused experience
 Lightweight, fast, and runs entirely in the browser — no installs, no sign-up
